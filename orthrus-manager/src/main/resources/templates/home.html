About the Application

Orthrus DAST is a comprehensive security tool designed to identify vulnerabilities in your web APIs. It operates dynamically, parsing your API definitions (like OpenAPI specifications) to discover all available endpoints, and then simulating real-world attacks to find weaknesses before attackers do.
