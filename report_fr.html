L'outil assigne un score CVSS de Base théorique qui peut ne pas refléter le contexte environnemental ou temporel de la vulnérabilité. Les auteurs et l'outil lui-même déclinent toute responsabilité en cas de vulnérabilité non détectée, de failles critiques manquées, ou de compromissions ultérieures des systèmes testés. Méthodologie et Critères d'Évaluation

Ce rapport présente les résultats d'une analyse automatisée de sécurité dynamique (DAST). L'analyse a consisté à envoyer divers payloads malveillants et des entrées inattendues à l'API cible afin d'identifier les vulnérabilités de sécurité.

Orthrus utilise une taxonomie avancée pour catégoriser ses découvertes.
  • Score CVSS 3.1 Base : Fournit un score numérique théorique (0.0 à 10.0) reflétant les qualités intrinsèques de la vulnérabilité.
  • CAPEC : Common Attack Pattern Enumeration and Classification, identifiant le mécanisme exact de l'attaque.
  • OWASP : Lie chaque vulnérabilité au classement de sécurité API Top 10 le plus récent.
  • Note de Sécurité Globale : Une note (de A à F) évaluant la posture globale de sécurité, basée sur la vulnérabilité de plus haute criticité trouvée.

1. OS Command Injection [CRITIQUE]
Catégorie OWASP: Injection | Confiance: HIGH
Score CVSS de Base: 9.8 78 CAPEC-88
CWE CWE-78: Neutralisation incorrecte des éléments spéciaux utilisés dans une commande système ('OS Command Injection')
Point de Terminaison GET https://petstore3.swagger.io/api/v3/pet/findByStatus
Description Le point de terminaison exécute des commandes fournies par l'utilisateur sur le système d'exploitation sous-jacent.
Remédiation Évitez d'exécuter des commandes système directement. Si nécessaire, utilisez des API de bas niveau sûres et validez rigoureusement les entrées (allowlist).
Preuve
Response contains the injected marker string 'VULNAPI_CMD_INJECT_SUCCESS', indicating the OS command was executed and its output was returned.
2. Server-Side Template Injection (SSTI) [CRITIQUE]
Catégorie OWASP: Injection | Confiance: HIGH
Score CVSS de Base: 9.8 1336 CAPEC-137
CWE CWE-1336: Neutralisation incorrecte des éléments spéciaux utilisés dans un moteur de template
Point de Terminaison POST https://petstore3.swagger.io/api/v3/pet/42
Description Le point de terminaison évalue l'entrée utilisateur comme du code de template, permettant l'exécution de code arbitraire sur le serveur.
Remédiation N'utilisez jamais d'entrées utilisateur directement dans l'évaluation d'un template. Utilisez des moteurs de template sans logique ou exécutez-les dans un environnement bac à sable sécurisé.
Preuve
Response contains the evaluated result ('49') of the injected template expression '{{7*7}}'.
